
# CTX Bot

Chatbot for Click To Whatsapp Ads




## Installation

Clone this repo on your machine and install the dependecies

```bash
  npm install
```




    
## Replace variables

Replace all these variables with the information generated on the documentation:

File index.js
    AD_ID

File ctxApi.js
    API_KEY
    GOAL_ID
    PAGE_ID
    DATA_SET_ID
    ACCESS_TOKEN
